# Anlyx Brand Assets

This directory stores the selected Anlyx brand marks.

- `anlyx-logo.png` — primary horizontal logo for README, docs, website headers, and package visuals.
- `anlyx-icon.png` — standalone icon for favicon, app buttons, drawer launchers, GitHub avatars, and compact UI.

The current mark uses a connected node path with a yellow decision point to represent Anlyx's core idea: tracing a real app action through the backend flow.
