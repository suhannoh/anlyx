# Anlyx Brand Assets

This directory stores the selected source Anlyx brand marks.

- `anlyx-logo.png` — source horizontal logo export.
- `anlyx-icon.png` — source standalone icon export.

Use the packaged derivatives in `docs/assets/brand/` for product and documentation surfaces:

- `anlyx-logo-card.png` — background-included horizontal logo for README and release pages.
- `anlyx-icon-card.png` — background-included icon for release visuals.
- `anlyx-logo-transparent.png` — transparent horizontal logo for light/dark layout composition.
- `anlyx-icon-transparent.png` — transparent icon for app buttons, drawer launchers, favicons, and compact UI.

The current mark uses a connected node path with a yellow decision point to represent Anlyx's core idea: tracing a real app action through the backend flow.
