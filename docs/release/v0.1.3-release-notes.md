# Anlyx v0.1.3

Anlyx v0.1.3 makes the project usable as an injected local developer tool.

The real frontend keeps running on its own localhost port, Anlyx runs behind it on port `4777`, and the Flow Drawer focuses on the API request caused by the user's latest action instead of behaving like a full network log console.

## Install

```bash
npm install -D anlyx
npx anlyx init
npx anlyx dev
```

## Highlights

- Inject Mode keeps the real frontend at `frontend.baseUrl` while Anlyx serves local runtime assets and report data from port `4777`.
- `AnlyxDevOverlay` injects the overlay only during Next.js development builds.
- The Flow Drawer prioritizes the latest click, submit, or key-triggered API request.
- Background auth, session, health, and polling requests are recorded but separated from the main flow.
- The matched backend flow is rendered as a React Flow diagram with fit/reset controls.
- The compact draggable launcher uses the Anlyx icon and avoids blocking the host app.
- React SPA and manual integration remain available through the raw overlay script.
- README and release assets show the actual Anlyx Flow Drawer component instead of a static mock.
- Packaging now resolves workspace dependencies correctly for published npm packages.

## Compatibility

Deep Support:

- Spring Boot backend endpoint and flow scanning
- Next.js App Router page discovery and capture

Basic Support:

- OpenAPI endpoint import
- Manual frontend URLs
- Manual overlay script injection for non-Next React apps

Not included in v0.1:

- FastAPI, Express, NestJS, or React Router Deep Support
- Java agent runtime tracing
- OpenTelemetry tracing
- Mermaid, PNG, or SVG export
- LLM-generated flow summaries

- Use `anlyx@0.1.3` or newer.
- Older `anlyx@0.1.0` and `anlyx@0.1.1` builds are not recommended.
