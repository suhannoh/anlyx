# Anlyx v0.1.6-beta.1 Release Notes

Anlyx v0.1.6-beta.1 is a beta hotfix for the trust-first Project JSON viewer.
It keeps the v0.1.6-beta.0 validation, coverage, source-reference, and agent
prompt improvements, and fixes the published viewer package so it renders
correctly when installed inside another React project.

## What Changed

- Moves React and React DOM out of `@anlyx/ui` runtime dependencies and treats
  them as peer dependencies.
- Adds React and React DOM to the `anlyx` CLI package so the standalone local
  viewer has one runtime copy when installed from npm.
- Updates setup prompts and public docs to use `npm install -D anlyx@beta`
  during the beta release window.
- Keeps `latest` on the stable v0.1.5 line until the beta is promoted.

## Install

```sh
npm install -D anlyx@beta
npx anlyx prompt init
npx anlyx validate anlyx.project.json
npx anlyx import anlyx.project.json
npx anlyx dev
```
